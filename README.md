# rsync-backup_almalinux

An incremental backup with rsync based on the script of Dr. Stefan Konarski https://stefankonarski.de/sysadmin/backup/inkrementelles-backup-mit-rsync.

### Usage

Run this script from a live system or another external system.

#### backup your system

`./rsync-backup_almalinux.sh backup`

#### restore your system

`./rsync-backup_almalinux.sh restore`
